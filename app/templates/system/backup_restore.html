数据库备份与恢复

{% if database_backend == 'mysql' %}

当前系统配置使用 MySQL 数据库，内置的备份与恢复按钮不可用。请在服务器上使用以下命令完成操作：

  1. 备份： 
    mysqldump -h <host> -u <user> -p'<password>' <database> > library-$(date +%F).sql
    将生成的 .sql 文件妥善保存或同步到备份存储。
  2. 恢复： 
    mysql -h <host> -u <user> -p'<password>' <database> < library-YYYY-MM-DD.sql
    如需在 Docker 环境中执行，可使用 docker compose exec db ... 在数据库容器内运行上述命令。
  3. 建议在维护窗口执行，并在恢复前后通过 SHOW TABLES;、SELECT COUNT(*) 等语句或应用自检确认数据完整性，同时记录操作日志。
  4. 确保部署环境中的 SQLALCHEMY_DATABASE_URI 等连接信息与实际的 MySQL 实例保持一致，以便应用在恢复后正常工作。
